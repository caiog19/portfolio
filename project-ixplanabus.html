iXplanabus

O iXplanabus é um projeto de um hackaton que tem como objetivo melhorar a mobilidade urbana no Rio de Janeiro, oferecendo rotas de ônibus otimizadas e Permite também o registro de incidentes em tempo real (buracos, acidentes, falta de luz, alagamentos, obras, etc.).

Tecnologias Utilizadas

  • React.js
  • Node.Js
  • Google API(Maps JS, Places, Direction)
  • DataRio API
  • Cloudinary
  • PostgreSQL
